HOSTING Y GITHUB

HTML - Desarrollo Web

¿Qué es el Hosting?

El hosting es un servicio que permite publicar sitios web en Internet. Se refiere al espacio en un servidor donde se almacenan archivos, imágenes, bases de datos y todo lo necesario para que un sitio funcione.

Algunos tipos de hosting son:

  • Hosting compartido
  • VPS (Servidor Privado Virtual)
  • Hosting dedicado
  • Cloud Hosting

¿Qué es GitHub?

GitHub es una plataforma para alojar código, basada en el sistema de control de versiones Git. Permite colaborar en proyectos, llevar historial de cambios y desplegar sitios web con GitHub Pages.

GitHub se usa comúnmente para:

  • Compartir código fuente
  • Colaborar en proyectos de software
  • Control de versiones
  • Publicar sitios estáticos con GitHub Pages

Proceso de creación de una página en HTML

  1. Crear un archivo con extensión .html
  2. Escribir la estructura básica con etiquetas <html>, <head>, y <body>
  3. Agregar contenido como títulos, párrafos, listas, imágenes, etc.
  4. Guardar el archivo y abrirlo en un navegador para visualizarlo

Ejemplo básico:

<!DOCTYPE html>
<html>
<head>
    <title>Mi Página</title>
</head>
<body>
    <h1>Hola mundo</h1>
</body>
</html>

Publicar tu sitio HTML con GitHub Pages

  1. Crear un repositorio en GitHub
  2. Subir tu archivo index.html
  3. Ir a la configuración del repositorio
  4. Activar GitHub Pages desde la rama main
  5. Tu sitio estará disponible en https://tuusuario.github.io/nombre-repositorio
